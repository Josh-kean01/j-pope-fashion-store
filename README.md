# J-Pope Storefront

Luxury fashion storefront frontend for J-Pope, built with React, Vite, and Tailwind CSS.

## Overview

This project is a polished ecommerce-style frontend concept with:

- editorial landing and lookbook pages
- product listing and product detail flows
- wishlist, cart, and checkout mock journey
- support, legal, and 404 pages
- route-level SEO metadata
- Vercel-ready SPA routing

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- ESLint

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deployment

This repo includes `vercel.json` for SPA rewrites on Vercel.

## Notes

- This is a frontend portfolio/MVP storefront, not a fully integrated commerce backend.
- Checkout, contact, and account flows are mock frontend experiences unless connected to real services.
